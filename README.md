echo "# Nepal Municipalities Data

Complete dataset of all 753 municipalities in Nepal, organized geographically from Jhapa (east) to far western districts.

## Data Structure
- `municipality_id`: Unique identifier (1-753)
- `name`: English name of municipality
- `nepali_name`: Nepali name of municipality 
- `district_id`: District identifier (1-77, ordered east to west)
- `local_level_type_id`: 1=Metro, 2=Sub-Metro, 3=Municipality, 4=Rural Municipality

## Usage
This JSON file can be used for:
- GIS mapping
- Data analysis
- Administrative applications
- Research on Nepal's local governance

## Source
Data compiled from official Nepal government sources.
